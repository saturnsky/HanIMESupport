한 입력기 소개

한 입력기는 iPhone과 iPad에서 편리한 한글 입력을 위해 설계된 다기능 키보드입니다. 다양한 한글 입력 방식과 영문 레이아웃을 지원하여, 사용자의 취향과 상황에 맞는 최적의 입력 환경을 제공합니다.

편의 기능

  • 세로/가로 모드별 맞춤 설정

사용 가이드

1. 키보드 설치

  1. App Store에서 한 입력기를 다운로드
  2. iOS 설정 > 일반 > 키보드 > 키보드로 이동
  3. "새로운 키보드 추가..." 선택
  4. "한 입력기" 선택
  5. "전체 액세스 허용" 활성화 (선택사항)

2. 자판 설정

  1. 한 입력기 앱 실행
  2. 세로 모드와 가로 모드별로 원하는 자판 선택
  3. 한글/영문 자판을 각각 설정
  4. 키보드 높이 및 햅틱 피드백 조정

자주 묻는 질문

키보드가 나타나지 않아요

iOS 설정 > 일반 > 키보드에서 한 입력기가 추가되어 있는지 확인하세요. 텍스트 입력 시 지구본 아이콘을 길게 눌러 키보드를 선택할 수 있습니다.

만약 정상적으로 설정을 마쳤는데도 키보드가 나타나지 않는다면, 핸드폰 재부팅이 필요한 경우가 있습니다.

자판을 변경하려면?

한 입력기 앱을 실행하여 설정에서 원하는 자판을 선택하세요. 세로/가로 모드별로 다른 자판을 설정할 수 있습니다.

한글이 제대로 입력되지 않아요

한 입력기 앱에서 원하는 한글 입력 방식이 선택되어 있는지 확인하세요. 세로 모드와 가로 모드는 각각 다른 설정을 가질 수 있습니다.

만약 설정이 정상적으로 되어 있는데도 문제가 발생하면, 버그 신고를 통해 신고해 주세요.

한손 모드는 어떻게 사용하나요?

키보드에서 좌측 또는 우측으로 스와이프하면 한손 모드로 전환됩니다. 두 손가락으로 양쪽으로 펼치면 원래 모드로 돌아갑니다.

플립 기능을 지원하는 키에서는 위의 제스처가 동작하지 않습니다. 그럴 때는 스페이스 등 플립 기능을 지원하지 않는 키에서 시도해 주세요.

자판의 동작이 예상과 다릅니다

일부 자판(모토로라, 베가 등)은 현재 해당 입력 방법을 사용하는 실제 기기를 구하기 어려워, 공개된 자료와 문서를 바탕으로 구현되었습니다.

실제 사용하셨던 자판의 동작과 차이가 있을 경우, 구체적인 동작 방식과 참고 자료를 포함하여 문의해 주시면 검토 후 개선하겠습니다.

단모음Next란 무엇인가요?

단모음Next는 기본 단모음 자판을 수정한 자판입니다. 기본 단모음 자판에서는 특정 조합 시 모호성으로 인해 입력이 어려운 단어들이 있어, 이를 해결하기 위해 조합 확정 버튼을 추가했습니다.

글자 자동 확정 기능에 의존하지 않고도 더 명확하고 일관된 입력이 가능합니다.

왜 전체 액세스를 요구하나요?

iOS 키보드 익스텐션에서 햅틱 피드백(진동) 기능을 제공하려면 전체 액세스 권한이 필요합니다. 이는 Apple의 iOS 시스템 제약 사항입니다.

전체 액세스를 허용하지 않아도 기본적인 입력 기능은 모두 사용할 수 있으며, 전체 액세스를 허용해도 개인정보나 입력 내용은 일체 수집하지 않습니다.

개인정보가 수집되나요?

한 입력기는 사용자의 개인정보를 일체 수집하지 않습니다. 모든 설정은 기기에만 저장되며, 입력 내용은 어디에도 전송되지 않습니다.

후원금은 어떻게 사용되나요?

후원해 주신 금액은 주로 애플 개발자 계정 유지 비용(연간 $99)으로 사용됩니다. 이를 통해 한 입력기를 지속적으로 App Store에서 제공하고 업데이트할 수 있습니다.

남은 금액이 있다면 추가 기능 개발이나 테스트 기기 구입 등 앱 개선을 위해 사용됩니다.

문의 메일에 답변이 없습니다

보내주신 모든 메일을 확인하고 있으나, 한 입력기는 개인 취미 프로젝트로 운영되고 있어 관리에 많은 자원을 투입하기 어려운 상황입니다.

이로 인해 모든 메일에 개별 답변을 드리기 어려운 점 양해 부탁드리며, 긴급한 버그나 중요한 문의사항의 경우 우선적으로 검토하고 있습니다.

새로운 자판을 추가해 주세요

한 입력기는 대부분의 한국어 자판으로 확장 가능하게 설계되어 있으나, 개발 자원의 한계로 요청하신 모든 자판을 추가하기는 어렵습니다.

자판 추가 요청 등을 통해 새 자판에 대한 수요를 파악하고 있으며, 충분한 수요가 확인될 경우 우선적으로 추가를 진행합니다.

단, 내부 사전 기반 자동 보정이 필요한 입력기와 슬라이드 입력 방식은 프로젝트 방침상 지원하지 않습니다.

새로운 기능을 추가해 주세요

기능 요청은 수요 조사 목적으로 활용하고 있으며, 충분한 수요가 확인될 경우 개발을 검토합니다.

다만 기능의 개발 난이도, 기술적 제약사항, 앱의 전체적인 설계 방향 등을 종합적으로 고려하여 개발 여부를 결정하므로, 수요가 많다고 해서 모든 기능이 구현되는 것은 아닙니다.
